Javascript String Methods

These are string methods that allow you to manipulate and work with text strings. Some of the methods include the below;

  1. To get the length of strings, use dot length
  2. To CAPITALIZE strings, use dot toUpperCase()
  3. To LOWERCASE strings, use dot toLowerCase()
  4. To replace characters in a string (with others), use dot Replace all
  5. To REPEAT STRING A CERTAIN NUMBER OF TIMES
  7. To check whether a string STARTS with certain characters, use dot StartsWith
  8. To check whether a string ENDS with certain characters, use dot EndsWith
  9. To Return a character at a specified index, use dot charAt()
  10. To Return the index of the FIRST Occurence of the specified character, use dot indexOf()
  12. To Return the index of the LAST Occurence of the specified character, use do lastIndexOf()
  14. To remove trailing and leading spaces use, dot Trim()
  15. To Pad a string at the front with characters, use dot padStarts(num of xters, xter to pad with)
  16. To Pad a string at the end with characters, use dot padEnds(num of xters, xter to pad with)

String Slicing

This is the process of creating a substring from the portion of another string. For instance, 'Flag Bearer' becomes 2 words; Flag and Bearer. Can be done using 'string.slice(start,end)'
